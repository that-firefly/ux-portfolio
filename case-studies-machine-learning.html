Visualising Data from Sentiment Analysis

In my daily UX practice, I utilise tools like AWS Amazon Comprehend whenever feasible and aligned with business requirements to analyse text in real-time. This includes extracting key phrases, detecting dominant languages, analysing sentiment, and identifying syntax. Amazon Comprehend, powered by AWS, is a Natural Language Processing (NLP) tool that uses machine learning to uncover insights and relationships within text.

For example, starting with an online review, such as one from BT, we can quickly extract key sentiments and insights using Amazon Comprehend, as shown in the example below. Key insights can be uncovered by:

  • Extracting key phrases
  • Analysing entity insights
  • Identifying key sentiments
  • Gaining insights from targeted sentiment
