Upload packaging data to your organisation’s account

Before you can submit your 2023 data file to the regulator, you need to upload it to your organisation’s account.

When you upload the file we’ll check your data is in the right format - this is called ‘validation’..

If the data is OK, the file will be stored on your organisation’s account. Only one file can be stored on the account.

You can update the file and upload it again as often as you need to before you submit it.

If there’s a problem with your file, we’ll reject the upload and tell you how to fix it.

Before you start

Read the guidance on how to upload packaging data as a compliance scheme (opens in a new window).
