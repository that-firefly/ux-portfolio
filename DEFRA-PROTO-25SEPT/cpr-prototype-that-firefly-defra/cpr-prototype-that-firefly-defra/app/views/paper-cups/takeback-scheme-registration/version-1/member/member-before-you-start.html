Uploading details about your members

As a takeback scheme you will need to register information about your members.

There are 2 spreadsheets you must upload.

1. Members organisation details

You’ll need to upload a spreadsheet which includes information about their:

  • organisation name and registered address
  • contact details
  • partners if they are in a partnership
  • main standard industrial classification (SIC) code

View the spreadsheet template for members organisation details.

2. Members premises

You’ll need to provide details about the individual premises for each member.

View the spreadsheet template for members premises.
