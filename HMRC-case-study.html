HMRC - His Majesty's Revenue and Customs case study

As part of the Advance Ruling Service (ARS) project for HMRC, I collaborated with a team of UX Designers, UX Researchers, Service Designers, and Web Developers to design and refine online pages. Our work involved updating an interactive prototype using GOV Design System components and GitHub, ensuring strict adherence to GDS design standards.

By connecting the GitHub prototype repository to a local development environment, I utilised HTML and JavaScript for quick, iterative updates based on real-time user feedback. These changes were tested directly with users, allowing us to refine the experience efficiently. Throughout this process, I leveraged Git Bash and Visual Studio Code to manage and deploy code seamlessly.
